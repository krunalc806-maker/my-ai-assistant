# 🤖 My AI Assistant

A modern AI assistant built with Next.js, Assistant UI, OpenRouter, and Google Authentication.

## ✨ Features

- 💬 AI Chat Interface
- 🤖 OpenRouter AI Integration
- 🔐 Google Sign-In (NextAuth)
- 🧠 Custom AI Identity
- ⚡ Streaming Responses
- 🎨 Modern Assistant UI
- 📱 Responsive Design

## 🛠️ Tech Stack

- Next.js 16
- React
- TypeScript
- Assistant UI
- OpenRouter API
- NextAuth
- Tailwind CSS

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/my-ai-assistant.git
cd my-ai-assistant
```

### Install dependencies

```bash
npm install
```

### Create `.env.local`

```env
OPENROUTER_API_KEY=your_openrouter_api_key
AUTH_GOOGLE_ID=your_google_client_id
AUTH_GOOGLE_SECRET=your_google_client_secret
AUTH_SECRET=your_random_secret
```

### Run the project

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
